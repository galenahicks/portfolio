cinemusician website

This web project is called "Cinemusician", in which the user is able to create their own composition of one of a song while simultaneously creating their own visual composition. Each of the sliders corresponds to a specific instrument part and video layer, showing the connection and similarities between the creation of visual art and music. Desktop version only; works best with Chrome
